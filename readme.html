Known Limitations

The following features, requested within the plugin's support thread, are not currently supported:

  • Integration with the "Dual Pricing" plugin (zc158a) and "Wholesale Pricing" feature (zc200+). Only retail pricing is provided by the plugin.

What it does

This encapsulated plugin provides, as its name implies, a printable price-list that your customers can view and/or print. Once the plugin's files are installed, you can configure your store's price list display using the new configuration screens created by the plugin's installation.

The plugin has been validated with:

  1. Zen Cart 1.5.8a
  2. Zen Cart 2.0.0
  3. Zen Cart 2.0.1
  4. ZCA Bootstrap Template, v3.7.0

Refer to the plugin's online wiki articles for additional information.

Note: If you are upgrading from a version prior to v4.0.0, you'll need to remove the files associated with that previous version of the plugin. See this GitHub wiki article for details.

Note: If you are using Zen Cart 1.5.8a or 2.0.0 (or a version of the ZCA Bootstrap Template prior to v3.7.0, you'll need add the plugin's required notification to enable its page-link addition to the Information sidebox. See this GitHub wiki article for details.

Installation

The plugin's installation, since it's an encapsulated plugin, is very easy.

  1. Unzip the plugin's package file, which you've most likely already done if you're reading this!
  2. Copy the /zc_plugins sub-directory and all its contents to the root directory of your Zen Cart file-system.
  3. Sign into your Zen Cart admin console.
  4. Navigate to Modules :: Plugin Manager and click on the Printable Price List row.
  5. From the right-hand sidebox, click the Install button and, from the next sidebox display click that Install button to install the plugin.

That's it! Next, check out the plugin's online wiki articles for additional information (like how to configure it).
